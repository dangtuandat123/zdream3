# UI_GUIDE.md

Mục tiêu: hướng dẫn UI để đảm bảo nhất quán.

## 1) Hướng phong cách
- Sạch, hiện đại, ưu tiên ảnh là trung tâm.
- Tối giản thao tác; tránh lộ thuật ngữ kỹ thuật.

## 2) Typography (đề xuất)
- Heading: font display (vd: "Sora" hoặc "Space Grotesk").
- Body: font dễ đọc (vd: "Manrope").

## 3) Color system (đề xuất)
- Primary: xanh biển đậm.
- Accent: cam hoặc teal để nổi bật CTA.
- Background: gradient nhẹ, tránh nền phẳng đơn điệu.

## 4) Layout
- Studio là trọng tâm: ảnh preview lớn, controls rõ ràng.
- Mobile-first, padding rộng, tap target >= 44px.

## 5) Motion
- Staggered reveal cho card style.
- Progress/polling state rõ ràng (skeleton + stepper).

## 6) Components bắt buộc
- Style card
- Studio controls (options, ratio, upload, advanced)
- Wallet card + QR block
- History grid

## 7) Accessibility
- Contrast >= 4.5:1.
- Form validation rõ ràng.
